Илл.3 «Audi music interface»: кабель-адаптер
Относится к автомобилям: с «Audi music interface» и кабелем-адаптером для iPod
Условие: разделительный подлокотник откинут Илл.2.
Подключить кабель-адаптер для iPod* -6- Илл.3 к iPod.
Для отсоединения iPod-штекера снять блокировку одновременным нажатием кнопок -1- Илл.1 и осторожно извлечь штекер из iPod -2-.
Аккумулятор iPod заряжается через кабель-адаптер для iPod* сразу после подключения iPod к «Audi music interface» и включения зажигания. Важно помнить, что вначале к «Audi music interface» необходимо подсоединить кабель-адаптер, а затем – подключить портативное воспроизводящее устройство.
К кабелю-адаптеру для iPod* можно подключать следующие iPod:
  • iPod mini – (начиная с версии 1.3.0);
  • iPod nano
  • iPod 4G – четвертое поколение (начиная с версии 1.2.1);
  • iPod 5G – пятое поколение видео;
  • iPod 6G – шестое поколение classic;
  • iPod touch;
  • iPhone.
Прочие версии iPod (например, iPod shuffle) нельзя подключить при помощи кабеля-адаптера для iPod*. С помощью предлагаемого в качестве опции кабеля-адаптера с коммутаторным штекером 3,5 мм* можно подключить данные устройства в качестве портативных воспроизводящих устройств Ссылка.
«Audi music interface» не поддерживает такие функции управления iPod, как оценка музыкальных композиций, пролистывание/отображение обложек, воспроизведение видеофайлов или добавление файлов в список воспроизведения «On-the-go».
Примечание
  • Для подключения iPod использовать только кабель-адаптер с номером 4F0 051 510 K.
  • Для воспроизведения видеофайлов подключить iPod с помощью опционального А/В-кабеля-адаптера* Ссылка к «Audi music interface». Чтобы подключить iPod к А/В-кабелю-адаптеру*, требуется дополнительный А/В-кабель Apple. Подробную информацию об А/В-кабеле Apple см. в руководстве по эксплуатации iPod.
  • Возникновение неисправностей в работе iPod сказывается также на MMI. В данных случаях выполнить перезагрузку iPod (см. руководство по эксплуатации iPod).
  • Важные пояснения по пользованию iPod см. в руководстве по эксплуатации iPod.
  • Мы рекомендуем позаботиться о новейшей версии программного обеспечения iPod.
