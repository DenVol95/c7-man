Audi MMI   Коммуникация   Пользование телефоном  Другие настройки для устройства подготовки мобильного телефона
После срабатывания подушки безопасности происходит автоматический набор номера экстренной помощи через подсоединенный мобильный телефон.
Условие: MMI Ссылка и устройство подготовки мобильного телефона Ссылка находятся в рабочем режиме.
Выбрать: функциональную клавишу TEL > клавишу управления Настройки > Настройки телефона > Авт. экстр. вызов.
Вкл.
После срабатывания подушки безопасности происходит автоматический набор номера экстренной помощи 112 через подсоединенный мобильный телефон. Точное местоположение автомобиля дополнительно определяется посредством функции идентификации радиосоты.
Выкл.
Автоматический набор номер экстренной помощи может быть выключен. Но после отключения двигателя функция набора экстренной помощи снова устанавливается на настройку по умолчанию Вкл..
ВНИМАНИЕ!
Следует помнить, что выполнение автоматического набора номера экстренной помощи не может быть гарантировано для любых возможных обстоятельств. Нельзя полагаться исключительно на телефонную связь в жизненно важных ситуациях (например, при необходимости экстренной медицинской помощи).
Примечание
Функция автоматического набора номера экстренной помощи доступна для всех стран ЕС, а также для Швейцарии, Норвегии, Лихтенштейна и Исландии.
