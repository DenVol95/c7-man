Относится к автомобилям: с автомобильным телефоном или устройством подготовки мобильного телефона
Через редактор или панель управления «MMI touch»* можно напрямую вводить телефонный номер в виде последовательности чисел/букв.
Условие: включен режим телефона Ссылка и выбрана функция Ввести номер -5- Илл.3.
Ввод телефонного номера посредством редактора
Ввести номер телефона через редактор Ссылка.
Для ввода номера телефона в виде последовательности букв нажать клавишу управления A-Z.
Ввести через редактор соответствующую комбинацию букв, напр., AUDISERVICE, Илл.2.
Подтвердить ввод поворотом редактора на OK и его последующим нажатием.
Для подтверждения ввода можно также нажать клавишу управления набрать. после установления соединения отключается звук включенного аудио-/видеоисточника.
Ввод телефонного номера посредством панели управления «MMI touch»*
Пальцем ввести с панели управления «MMI touch*» телефонный номер/последовательность букв. Однозначно распознанный знак отображается на дисплее MMI и дублируется речевым сообщением.
Для выбора альтернативного знака вращать кнопку управления влево/вправо. Выбранный знак переносится в поле ввода.
Для завершения ввода коротко нажать на панель управления «MMI touch*». Введенный номер телефона/последовательность букв набирается.
Чтобы удалить сразу все неправильно введенные знаки, вращением редактора выбрать и удерживать кнопку управления нажатой. Для удаления знака с помощью панели управления «MMI touch*» провести пальцем влево по «MMI touch*».
Примечание
  • Во время сеанса связи сигналы DTMF можно вводить посредством редактора.
  • При выключенной функции Нав. сообщение при тел. разговоре в навигационной системе инструкции по навигации во время сеанса связи не воспроизводятся Ссылка.
  • Выключенный дисплей MMI автоматически включается при поступлении телефонного вызова.
