Выбрать: функциональную клавишу MENU > клавишу управления Настройка MMI > Программное обновление системы.
С помощью функции «Обновление системы» возможно обновление программного обеспечения MMI (напр., навигационные данные, электронная версия руководства по эксплуатации, банк данных Gracenote®) с DVD, карты SD или флэш-накопителя.
Примечание
Дополнительная информация о функции Программное обновление системы приведена на странице www.audi.de/mp3 (www.audi.com/mp3).
