Илл.1 Ввод навигационной цели
Илл.2 Сохранение радиостанций
Посредством панели управления «MMI touch» возможна работа со многими важными функциями, например ввода букв/чисел.
Ввод букв, чисел и знаков
Пальцем изобразить знак на панели управления «MMI touch» Ссылка. Однозначно распознанный знак отображается на дисплее MMI -1- Илл.1 и дублируется речевым сообщением.
Для выбора альтернативного знака -2- вращать кнопку управления влево/вправо. Выбранный знак переносится в поле ввода -3-.
Для ввода пробела провести пальцем по панели управления слева направо.
Для удаления знака провести пальцем по панели управления справа налево.
Для завершения ввода коротко нажать на панель управления.
Перемещение карты/настройка центра звучания
Нажать кнопку управления. На карте появляется перекрестие.
Для смещения перекрестия в вертикальном направлении провести пальцем по панели управления вверх/вниз.
Для смещения перекрестия в горизонтальном направлении провести пальцем по панели управления влево/вправо.
Для смещения перекрестия в диагональном направлении провести пальцем по панели управления вверх и влево/вправо либо вниз и влево/вправо.
Сохранение/выбор радиостанции
Для выбора сохраненной радиостанции коротко нажать на нужное число (1-6) на панели управления. Кнопки радиостанций (1-6) появляются на дисплее MMI -1- Илл.2 и радиоприемник переключается на выбранную радиостанцию.
Для сохранения установленной радиостанции удерживать палец на нужном числе (1-6) на панели управления до звукового сигнала. Радиостанция сохраняется в выбранной позиции перечня памяти.
Пролистывание/выбор обложек альбомов
Проводить пальцем по панели управления влево/вправо. На дисплее происходит пролистывание обложек альбомов.
Для выбора нужного альбома коротко нажать на панель управления.
Управление в основном меню DVD-диска
Для выбора пункта меню проводить пальцем по панели управления вверх/вниз или влево/вправо.
Для подтверждения выбора меню коротко нажать на панель управления.
Кратковременным или продолжительным нажатием стрелок на панели управления «MMI touch» можно также перемещать перекрестие по карте, настраивать центр звучания, пролистывать обложки альбомов или работать с основным меню DVD.
На панели управления «MMI touch» могут отображаться следующие показатели состояния различных режимов ввода:
  • 1-6 = включен режим прямого выбора сохраненных радиостанций;
  • стрелки = выбрана функция Браузер альбомов Ссылка, включен режим перемещения центра звучания либо карты системы навигации.
До тех пор, пока на дисплее MMI отображаются кнопки радиостанций (1-6) -1- Илл.2, движением пальца по панели управления влево/вправо можно выбрать нужную радиостанцию/позицию перечня памяти.
Для перехода в адресной книге к начальной букве имени нужного контакта изобразить эту букву пальцем на панели управления. При вводе следующих букв индикация перечня найденных контактов становится точнее.
Если в адресной книге включить функцию Искать контакт, можно завершить ввод контакта и непосредственно перейти к списку выбора найденных контактов, коротко нажав на панель управления.
При использовании навигации можно завершить ввод (например, целевого города) и непосредственно перейти к списку выбора (например, найденных городов), коротко нажав на панель управления.
