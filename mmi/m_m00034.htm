Audi MMI   Знакомство с системой  Основные операции управления
В электронную версию руководства по эксплуатации входят руководство по эксплуатации конкретного автомобиля и руководство по эксплуатации MMI.
Условие: диск с электронной версией руководства по эксплуатации вставлен в дисковод для DVD Ссылка.
Инсталляция электронной версии руководства по эксплуатации
Нажать функциональную клавишу CAR.
Нажать клавишу управления рук. по эксп..
Для начала инсталляции нажать кнопку управления. На дисплее MMI появляются выбираемые источники.
Вращением кнопки управления выбрать CD/DVD и нажать кнопку управления.
Вращением кнопки управления выбрать начать загрузку и нажать кнопку управления. После успешного завершения инсталляции руководство по эксплуатации появляется на дисплее MMI.
Запуск электронной версии руководства по эксплуатации
Нажать функциональную клавишу CAR.
Нажать клавишу управления рук. по эксп.. Электронная версия руководства по эксплуатации отображается на том языке, который также был настроен в качестве языка меню Ссылка.
Обновление электронной версии руководства по эксплуатации
Нажать функциональную клавишу MENU.
Нажать клавишу управления Настройка MMI.
Вращением кнопки управления выбрать Прог. обновлен. системы Ссылка и нажать кнопку управления. На дисплее MMI появляются выбираемые источники.
Вращением кнопки управления выбрать CD/DVD и нажать кнопку управления.
Вращением кнопки управления выбрать начать загрузку и нажать кнопку управления. На дисплее MMI отображается продвижение процесса актуализации.
Для завершения обновления вращением кнопки управления выбрать OK и нажать кнопку управления.
Описание работы с электронной версией руководства по эксплуатации приведены в Ссылка.
ВНИМАНИЕ!
Невнимательность к дорожному движению может привести к серьезным авариям со смертельным исходом. Использовать электронную версию руководства по эксплуатации только в том случае, если это позволяет ситуация на дороге. Необходимо постоянно контролировать автомобиль. Водитель несет ответственность за безопасность движения.
Примечание
  • На время инсталляции/обновления электронной версии руководства по эксплуатации выключить зажигание.
  • Помнить, что электронная версия руководства по эксплуатации ни в коем случае не заменяет печатного издания руководства по эксплуатации.
