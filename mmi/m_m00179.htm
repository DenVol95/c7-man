Илл.1 Информация на дисплее SMS
Относится к автомобилям: с трубкой для автомобильного телефона
Текст для SMS можно вводить обычным способом или с помощью автоматического распознавания слов (Т9).
Переключение типа ввода текста
Во время ввода текста клавишей # можно переключаться между обычным режимом ввода текста, вводом текста с автоматическим распознаванием слов (Т9), вводом цифр, а также между верхним и нижним регистрами букв.
Ввод специальных символов и цифр
Для ввода специального символа удерживать клавишу * + нажатой и выбрать специальный символ посредством навигационной клавиши.
Для ввода цифры в буквенном режиме удерживать нажатой соответствующую клавишу с цифрой.
Обычный ввод текста
Для начала ввода слова нажать одну из клавиш 2 abc – 9 wxyz.
Нажимать соответствующую клавишу до тех пор, пока не будет показан нужный символ.
Для ввода пробела нажать клавишу 0.
Для удаления символа слева от курсора нажать клавишу управления Удалить Илл.1.
Ввод текста с помощью автоматического распознавания слов (Т9)
Для начала ввода слова нажать одну из клавиш 2 abc – 9 wxyz.
Нажимать для каждой буквы слова соответствующую клавишу только один раз, даже если соответствующая буква при вводе не отображается на дисплее.
Ввести все буквы слова.
Если в словаре содержится более одного варианта слова, выбрать нужное слово нажатием навигационной клавиши вверх/вниз.
Чтобы ввести пробел и, таким образом, подтвердить нужное слово, нажать клавишу 0.
Для удаления символа слева от курсора нажать клавишу управления Удалить.
Если слово в словаре отсутствует, слово можно вставить посредством клавиши управления Опции и функции Вставить слово. Слово перенимается в SMS и сохраняется в словаре.
Пояснение символов на рисунке Илл.1
-1-
Выбрана настройка автоматического распознавания слова T9.
-2-
Оставшиеся символы для SMS.
-3-
Введенные буквы (например, Audi) отображаются на сером фоне, пока не закончится ввод.
-4-
Указание по количеству вариантов слова (например, «1» память/«4» количество вариантов слова).
-5-
Установленный вариант написания прописных или строчных букв
Примечание
Длинные сообщения разделяются и посылаются по два сообщения или более. Длина сообщения ограничена 765 символами.
