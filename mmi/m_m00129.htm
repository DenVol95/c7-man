Audi MMI   Коммуникация  Адресная книга
Для существующего контакта адресной книги можно определить служебную и частную навигационную цель.
Условие: открыта адресная книга Ссылка, создается новый контакт Ссылка.
Вращением кнопки управления выбрать Служебная навигационная цель или Частная навигационная цель и нажать кнопку управления.
Вращением кнопки управления выбрать Cоздать навигационную цель и нажать кнопку управления. На дисплее MMI появляется перечень вариантов ввода цели.
Ввести цель Ссылка.
Вращением кнопки управления выбрать Перенять адрес и нажать кнопку управления. Навигационная цель вводится в память адресной карты.
Каждому контакту можно присвоить две цели: частную и служебную. Открытие сохраненных навигационных целей в режиме навигации осуществляется нажатием клавиши управления Навигация к цели Ссылка.
При наличии установленного частного/служебного почтового адреса Ссылка он автоматически переносится в поле ввода Адрес соответствующей навигационной цели. Впоследствии этот адрес может быть установлен в качестве навигационной цели простым выбором опции Перенять адрес.
В пункте меню важные особые цели перечислены ближайшие туристические автостоянки, заправочные станции, сервисные центры Audi или больницы. Пункт меню особая цель содержит несколько подменю, которые можно использовать для создания навигационных целей. Подробную информацию о подменю см. начиная с Ссылка.
Примечание
  • Навигационной цели может быть присвоен статус Показать как фаворита. В этом случае навигационная цель будет отображаться в категории Фавориты меню Навигация к цели режима навигации Ссылка.
  • Контакты, которым присвоена навигационная цель, сохраняются в локальной памяти открытой адресной книги.
