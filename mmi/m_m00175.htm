Илл.1 Информация на дисплее
Относится к автомобилям: с трубкой для автомобильного телефона
Нажать клавишу управления Меню -9- Илл.1 и выбрать с помощью навигационной клавиши Адресная книга.
Поиск контакта
При вводе начальной буквы контакта осуществляется переход в перечень контактов, начинающихся на эту букву.
Если контакт выбирается клавишей управления Ok, возможна реализация следующих функций:
  • при нажатии Детали отображаются фамилия, имя и номер телефона, которые можно обработать, выбрав функцию Изменить контакт;
  • при нажатии Удалить контакт выбранный контакт удаляется, а при нажатии Вызвать - создается телефонное соединение.
Новый контакт
Ввести имя и номер телефона, которые требуется сохранить, и нажать Сохранить контакт. Контакту может быть также присвоен статус Общий контакт Ссылка.
Порядок сортировки
Сортировка контактов в адресной книге осуществляется в алфавитном порядке - по фамилии или имени.
Общ. контакты
Отображение и скрытие контактов, сохраненных в общей локальной памяти MMI.
Сокращенный набор
Если для контакта создан и выбран номер для сокращенного набора, то при нажатии клавиши управления Ok возможна реализация следующих функций:
  • изменить контакт – изменение позиции телефонного номера в перечне сокращенного набора. Пролистать посредством навигационной клавиши и выбрать контакт, дважды нажав клавишу управления Ok. Контакт присваивается выбранной позиции;
  • удалить – удаление выбранного телефонного номера из перечня сокращенного набора.
Емкость памяти
Отображается Емкость памяти текущей адресной книги и общей памяти.
Удалить все конт.
  • При выборе Личные контакты удаляются все контакты личной локальной памяти MMI.
  • При выборе Публ. контакты удаляются все контакты общей локальной памяти MMI.
Примечание
Возможно удаление только контактов из локальной памяти MMI Ссылка.
