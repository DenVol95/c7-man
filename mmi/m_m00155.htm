Илл.1 Конфигурация канала передачи данных
Относится к автомобилям: с автомобильным телефоном и мобильным телефоном с профилем «SIM Access»
Для пользования онлайн-услугами через автомобильный телефон необходимо провести конфигурацию канала передачи данных.
Условие: включен режим пользования телефоном Ссылка.
Нажать клавишу управления Настройки.
Вращением кнопки управления выбрать Соединения и нажать кнопку управления.
Вращением кнопки управления выбрать Канал передачи данных и нажать кнопку управления. На дисплее MMI появляется меню Канал передачи данных.
В меню Канал передачи данных возможен выбор следующих пунктов настройки:
Установление связи
  • По запросу - канал передачи данных устанавливается по запросу.
  • Без запроса – канал передачи данных устанавливается автоматически при необходимости доступа к онлайн-услугам.
  • Нет - канал передачи данных не устанавливается. Доступ к онлайн-услугам невозможен.
При установленном канале передачи данных в статусной строке дисплея MMI отображается символ активного канала передачи данных -1- Илл.1.
Роуминг передачи данных
Использование канала передачи данных за границей возможно в том случае, если включен Роуминг передачи данных ().
Настройки соединения
В зависимости от используемой SIM-карты производится автоматическая конфигурация соединения. Данные значения можно изменить. Более подробную информацию можно получить у оператора мобильной связи.
  • APN - точка доступа зависит от оператора мобильной связи и присваивается автоматически;
  • Имя пользователя – имя пользователя для создания онлайн-профиля присваивается оператором мобильной связи.
  • Пароль - пароль для создания онлайн-профиля присваивается оператором мобильной связи.
  • Аутентификация - в зависимости от правил работы оператора мобильной связи может потребоваться аутентификация (подтверждение подлинности) по отношению к поставщику услуг. Если аутентификация требуется, выбрать настройку защищенная, во всех остальных случаях - стандарт.
Измененные настройки сохраняются только при выходе из меню Настройки соединения посредством кнопки BACK.
Сброс конфигурации
Выполняется создание и автоматическая конфигурация онлайн-профиля. Настройки соединения, выполненные вручную, сбрасываются на стандартные значения.
Прерывание активного канала передачи данных
Активный канал передачи данных закрывается. Для повторного установления канала передачи данных необходимо вызвать онлайн-услугу.
Примечание
  • Включение передачи данных за границей (Роуминг передачи данных) может привести к дополнительным расходам. Более подробную информацию можно получить у оператора мобильной связи.
  • Если в Настройках соединения заданы неправильные значения, канал передачи данных установить невозможно.
  • После разъединения связи «Bluetooth» между автомобильным и мобильным телефоном или извлечения SIM-карты из устройства считывания SIM-карт заданные вручную значения в Настройках соединения автоматически удаляются.
