Audi A6   Управление  Система индикации ограничения скорости
No speed limit detected (ограничение скорости не распознано)
Ограничение скорости не распознано (например, на автомагистрали без ограничения).
Индикация ограничения скор.: вне зоны действия
В навигационных данных отсутствует ограничение скорости для региона, в пределах которого движется автомобиль. Система не выдает ограничения скорости.
Индикация ограничения скорости: сейчас недоступна
Указание для водителя появляется при неисправности камеры или навигационной системы. Ограничения скорости определяются только одним источником, вследствие чего увеличивается вероятность ошибок.
Индикация ограничения скорости: сейчас нет доступа
Система временно не может обеспечивать функционирование и поэтому отключена.
Индикация ограничения скорости: системная неисправность!
Система больше не может обеспечивать функционирование и поэтому отключена. По возможности быстрее доехать до ближайшего специализированного предприятия.
Индикация ограничения скорости: сейчас недоступна. Камера не видит
Указание для водителя появляется, если поле зрения камеры закрыто Ссылка.
