Илл.1 Центральная консоль: кнопка системы помощи управления рулевым колесом
Относится к автомобилям: с системой помощи управления рулевым колесом при парковке
Водитель несет ответственность за процесс выезда со стояночного места.
Если автомобиль припаркован на стояночном месте, расположенном параллельно дороге, система помощи управления рулевым колесом при парковке может выполнить выезд из такого стояночного места.
Включить двигатель.
Нажать кнопку Илл.1.
Подождать, пока на дисплее появится указание для водителя Для выезда со стояночного места включить передачу заднего хода и указатели поворота.
Включить передачу заднего хода.
Включить соответствующий указатель поворота для выезда из стояночного места.
Убрать руки с рулевого колеса.
Следить за путем движения. Если дорожная ситуация позволяет, нажать на педаль акселератора с соответствующим усилием ВНИМАНИЕ!.
Следить за визуальной информацией на дисплее Ссылка и акустическими предупредительными сигналами до тех пор, пока выезд со стояночного места не будет выполнен.
Взять управление автомобилем на себя.
Система помощи при парковке автоматически управляет рулевым колесом при выезде со стояночного места, а водитель нажимает на соответствующие педали. Максимальная скорость при выезде со стояночного места составляет 5 км/ч. При вмешательстве в процесс маневрирования или при слишком высокой скорости система помощи управления рулевым колесом при парковке отключается.
Если работа системы помощи управления рулевым колесом при парковке была прервана, можно повторить процесс парковки автомобиля с помощью данной системы, начав выезд с места стоянки заново. Процесс выезда с места стоянки можно также продолжить самостоятельно, не включая систему помощи управления рулевым колесом.
