Илл.1 Подкапотное пространство: бачок стеклоомывателя
Бачок стеклоомывателя содержит жидкость для очистки ветрового стекла и для cистемы очистки фар* Илл.1. Заправочный объем бачка: Ссылка.
Чтобы избежать отложения извести в распылительных соплах, необходимо по возможности использовать чистую мягкую воду. Добавить в жидкость для омывания средство для чистки стекла (в зимний период – с антифризом).
Осторожно!
  • Ни в коем случае не примешивать антифриз, предназначенный для систем охлаждения, или иные добавки.
  • Не использовать средства для чистки стекол, содержащие растворитель лака – опасность повреждения лака.
