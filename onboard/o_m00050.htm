Audi A6   Управление  Информационная система для водителя
Илл.1 Многофункциональное рулевое колесо: управление информационной системой для водителя
Илл.2 Информационная система для водителя: вызов меню «Функции автомобиля»
Илл.3 Дисплей: пример информационной строки
Управление информационной системы для водителя осуществляется кнопками на многофункциональном рулевом колесе.
Принцип управления
Включить зажигание.
Для переключения закладок нажать двухпозиционный выключатель -1- влево/вправо Илл.1.
Для отображения прочей информации внизу/вверху повернуть валик меню -2- вниз/вверх.
Для подтверждения выбора нажать валик меню -2-.
Для открытия подменю активной закладки нажимать клавишу -3-.
Чтобы вызвать функцию программируемой клавиши рулевого колеса*, нажать клавишу -4-.
Вызов меню «Функции автомобиля»
Выбрать с помощью двухпозиционного выключателя -1- первую закладку.
Нажать кнопку -3-. Появляется меню Функции автомобиля Илл.2.
Чтобы выбрать пункт меню, повернуть и нажать валик -2-.
Установка нулевых значений
Выбрать в меню Функции автомобиля пункт Борткомпьютер или Прог. эффективности.
Теперь можно выбрать долговременное или кратковременное запоминающее устройство (ЗУ).
Чтобы сбросить значения соответствующего ЗУ на ноль, нажать валик меню -2- и удерживать его нажатым одну секунду.
Присвоение функции программируемой клавише рулевого колеса*
Выбрать: функциональную клавишу автомобиля CAR > кнопку управления (Car)* Системы > настройки автомобиля > раскладку клавиш рулевого колеса.
При включении зажигания отображается последняя выбранная функция.
Бортовой компьютер
В бортовом компьютере можно последовательно просмотреть следующую информацию:
  • дату;
  • время движения (ч) из кратковременного ЗУ;
  • средний расход (л/100 км) из кратковременного ЗУ;
  • или среднюю скорость (км/ч) из кратковременного ЗУ;
  • количество пройденных километров (км) из кратковременного ЗУ;
  • расход топлива на текущий момент (л/100км);
  • обзор кратковременного ЗУ;
  • обзор долговременного ЗУ.
Кратковременное запоминающее устройство сохраняет информацию о движении с момента включения зажигания до момента его выключения. При возобновлении движения в течение 2 часов после выключения зажигания к уже сохраненной информации о движении добавляются новые значения.
Информация в долговременном запоминающем устройстве, напротив, не удаляется автоматически. Благодаря этому можно устанавливать временный период, по которому производится анализ информации о движении.
При выборе некоторых функций автомобиля в строке -1- Илл.3 можно отобразить информацию о движении из кратковременного ЗУ.
Программа эффективности
Программа эффективности может помочь сэкономить топливо. Программа анализирует данные по расходу топлива и показывает дополнительные потребители Ссылка. Советы по экономичному вождению Ссылка содержат информацию об эффективной манере вождения. Информация о пройденном расстоянии и расходе топлива поступает в программу эффективности из бортового компьютера.
Для индикации программы эффективности выберите в меню Функции автомобиля пункт Прог. эффективности. При удалении данных из программы эффективности выполняется также сброс данных в бортовом компьютере.
Примечание
  • В некоторых экспортных исполнениях автомобиля единицы измерения могут отличаться.
  • При отсоединении клемм аккумулятора все сохраненные значения удаляются из запоминающего устройства.
