Илл.1 Дисплей: пример совета по экономичному вождению
Относится к автомобилям: с программой эффективности
При определенных условиях расхода топлива в программе эффективности автоматически кратковременно появляются советы по экономичному вождению. Соблюдая данные советы, можно снизить расход топлива.
Для выключения индикации совета по экономичному вождению непосредственно после его появления нажать любую кнопку управления на многофункциональном рулевом колесе.
Примечание
  • Выключенная индикация совета по экономичному вождению появится снова только после повторного включения зажигания.
  • Советы по экономичному вождению появляются не каждый раз, а через относительно большие временные интервалы.
