Audi A6   Самопомощь   Аварийная помощь  Помощь при запуске двигателя
В экстренном случае двигатель можно завести с помощью аккумулятора другого автомобиля.
Если двигатель не заводится, так как аккумулятор разряжен, то для запуска двигателя можно использовать аккумулятор другого автомобиля. Для этого необходим пусковспомогательный кабель.
Оба аккумулятора должны иметь номинальное напряжение 12 В. Емкость токоотдающего аккумулятора (измеряемая в ампер-часах) не должна быть намного ниже, чем у разряженного.
Пусковспомогательные кабели
В качестве пусковспомогательных кабелей можно использовать только кабели с достаточно большим сечением. Обращать внимание на параметры, приведенные заводом-изготовителем кабеля.
Использовать только пусковспомогательные кабели с изолированными полюсными зажимами.
Положительный кабель – чаще всего красного цвета.
Отрицательный кабель – чаще всего черного цвета.
ВНИМАНИЕ!
  • Разряженный аккумулятор может замерзнуть уже при 0 °C. Перед подключением пусковспомогательных кабелей к замерзшему аккумулятору дать ему обязательно оттаять. В противном случае он может взорваться.
  • При проведении работ в подкапотном пространстве помнить о предупреждениях Ссылка.
Примечание
  • Между автомобилями не должно быть контакта. Иначе цепь будет замыкаться уже при соединении плюсовых выводов.
  • Разряженный аккумулятор должен быть надлежащим образом соединен с бортовой сетью автомобиля.
  • При необходимости отключить автомобильный телефон. Более подробную информацию в данном случае можно получить из руководства по эксплуатации телефона.
